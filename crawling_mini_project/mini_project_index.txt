1조 김은비 최세윤 백영서 홍석준

주제 : 영화관 홈페이지 상영영화 순위 모음

담당
    - CGV 크롤링 (백영서)
    - 메가박스 크롤링 (김은비)
    - 시네큐브 크롤링 (홍석준)
    - 코드 통합 및 크롤링 값 html 형식으로 변경 (최세윤)

각 대형 영화관의 상영작 순위를(광고 제외) 크롤링하여 출력을 한다.
출력 방식은 html 파일을 제작하는 방식과 flask 서버로 실행하는 방식 두가지로 만든다
    - flask_server.py : 플라스크 서버를 적용한 방식
    - save_html.py : html 파일으로 저장하는 방식
html 코드에 .@media는 적용하지 않았음
